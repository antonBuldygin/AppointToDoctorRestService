Description

For better management hospital work director needs to get information about doctor’s workload. In this stage, let us create statistics of appointments quantity per doctor and  statistics of appointments quantity per day.

 

Objectives

At this stage implement two endpoints the /statisticsDay endpoint that handles GET requests and returns the information about the number of appointments set per day and the /statisticsDoc endpoint that handles GET requests and returns the information about the number of appointments set per doctor

 

Examples

Example 1: a GET  /statisticsDay    endpoint that returns statistics of  appointments per day. Example of json array response:

Response: 200 OK

[ {
  "2022-10-15" : 2
}, {
  "2022-10-14" : 1
}, {
  "2022-10-17" : 1
} ]

Example 2: a GET /statisticsDoc    endpoint that returns statistics of  appointments per doctor. Example of json array response:

Response: 200 OK

[ {
  "director" : 1
}, {
  "lea wong" : 1
}, {
  "pamela upperson" : 3
} ]

Example 3: a GET /statisticsDoc    if no appoints were set:

Response: 404 Not Found 

{
    "timestamp": "2022-10-30T11:15:48.181+00:00",
    "status": 404,
    "error": "Not Found",
    "message": "No message available",
    "path": "/statisticsDoc"
}

Response: 404 Not Found 

{
    "timestamp": "2022-10-30T11:15:48.181+00:00",
    "status": 404,
