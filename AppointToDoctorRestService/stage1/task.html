Description

In our hospital, there are more and more patients. The receptionist used to write down the visits on sticky notes. Unfortunately, the sticky notes get lost or taken away by doctors. The hospital director thinks that at the beginning a simple program for the receptionist will be sufficient. Let's make our program with the help of a REST service. In this stage, you need to create 3 endpoints that will return the information about appointments in JSON format.

Objectives

Implement the following endpoints: that handles GET requests and returns the information about the movie theatre.

  • /setAppointment that handles POST requests to create new appointment and returns the information about the appointment.
  • /deleteAppointment that handles DELETE requests to delete appointment and returns the information about the appointment deleted.
  • /appointments that handles GET requests and returns the information about all appointments.

    • The response should contain doctor name, patient name, date in the format "yyyy-MM-dd HH:mm" and appointment id number. The response is a JSON object and has the following format:

    { "idApp": 1, "doctorName": "Pamela Upperson ", "patientName": "Phill good", "date": "2021-12-01 22:00" }

    Examples

    Example 1: a POST /setAppointment request

    Response body:

    { "idApp": 1, "doctorName": "Pamela Upperson ", "patientName": "John Galt", "date": "2021-12-01 22:00" }

    Example 2: a DELETE /deleteAppointment ?id=1 request

    Response body with deleted appointment information:

    { "idApp": 1, "doctorName": "Pamela Upperson", "patientName": "John Galt", "date": "2021-12-01 22:00" }

    Example 3: a DELETE /deleteAppointment?id=1 request

    Response body if appointment does not exist with such id number:

    { "error": "The appointment was already cancelled or does not exist!" }

    Example 4: a GET /appointments request

    Response body with all created appointments:

    { "1": { "idApp": 2, "doctorName": "Pamela Upperson", "patientName": "John Galt", "date": "2021-12-01 12:00" }, "2": { "idApp": 3, "doctorName": "Lea Wong", "patientName": "Indi Grimes", "date": "2021-11-15 11:00" } }
